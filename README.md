# Mathi

A small math game for kids built with Elm

## Running it locally

Install dependendencies
```sh
npm install
```

Run it with a local web server

```sh
npm start
```

Build for production

```sh
npm run build
```

## Built with ...

- [Elm](https://elm-lang.org/)
- [Parcel](https://parceljs.org/)
- [TailwindCSS](https://tailwindcss.com)
